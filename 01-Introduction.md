# Python Introduction

## What is Python?

Python is a high-level, interpreted, and general-purpose programming language known for its simple syntax and readability.

It was created by **Guido van Rossum** and first released in **1991**.

Python is one of the most popular programming languages in the world and is widely used in:

- Web Development
- Data Science
- Artificial Intelligence
- Automation
- Cybersecurity
- Machine Learning
- Software Development

---

## Why Learn Python for Cybersecurity?

Python is one of the most widely used programming languages in cybersecurity because it allows security professionals to automate repetitive tasks and build custom tools.

Examples include:

- Port scanners
- Vulnerability scanners
- Password generators
- Log analyzers
- Web scrapers
- Packet analyzers
- Malware analysis tools

Many popular cybersecurity tools are built using Python.

Examples:

- Scapy
- Impacket
- Pwntools
- Volatility

---

## Features of Python

### 1. Easy to Learn
Python uses simple and readable syntax.

Example:

```python
print("Hello World")
```

---

### 2. Interpreted Language

Python code is executed line by line by an interpreter.

Unlike C or C++, Python does not require compilation before execution.

---

### 3. Cross Platform

Python programs can run on:

- Windows
- Linux
- macOS

without major changes.

---

### 4. Open Source

Python is free to use and has a large community of developers.

---

### 5. Large Standard Library

Python comes with many built-in modules and libraries.

Examples:

- os
- sys
- math
- random
- socket

---

### 6. Supports Multiple Programming Paradigms

Python supports:

- Procedural Programming
- Object-Oriented Programming
- Functional Programming

---

## Installing Python

Check if Python is installed:

```bash
python --version
```

or

```bash
python3 --version
```

Example output:

```text
Python 3.12.0
```

---

## Running Python

Start the Python interpreter:

```bash
python3
```

Example:

```python
>>> print("Hello Cybersecurity")
Hello Cybersecurity
```

Exit Python:

```python
exit()
```

or

```text
Ctrl + D
```

---

## Writing Your First Python Program

Create a file:

```text
hello.py
```

Code:

```python
print("Hello World")
```

Run the program:

```bash
python3 hello.py
```

Output:

```text
Hello World
```

---

## Python File Extension

Python files use the extension:

Examples:


## Python Interpreter vs Compiler

| Feature | Python | C |
|---------|--------|---|
| Type | Interpreted | Compiled |
| Execution | Line by Line | Entire Program |
| Development Speed | Fast | Slower |
| Syntax | Simple | Complex |
